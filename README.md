# BombermanOS
BombermanOS is software for cardputerADV, primarily designed for media playback and file display.

Currently, it can play MP3 files, and in the future, I hope to enable playback via Bluetooth.

Soon, features for viewing and editing text, as well as viewing images, are planned.





How To Install?
 option 1: 
 download latest bin file from release and install it via esp flash_download_tool
 how to use and download see here https://docs.espressif.com/projects/esp-test-tools/en/latest/esp32/production_stage/tools/flash_download_tool.html

  option 2: (recomended)
install it via M5Burner Select Cardputer in the sidebar and search for BombermanOS
see info here https://docs.m5stack.com/en/uiflow/m5burner/intro

option 3 
install  it via luncher 
if you have launcher install in your cardputer you can install it via sd card , download the latest bin file and put it in the sd card!

see info here https://github.com/bmorcelli/Launcher




How To Use?
vary simple!!  ESC key to go back!  Enter Key to select  -/+ keys to volume up or down
